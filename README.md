## 概要
[TORO乗換案内アプリ](https://github.com/TORO-Server/toroapp_transfar) をベースにビジュアルや機能を一新したTOROの交通をまとめたアプリです

## リリースノート
| バージョン     | 機能              | Web    | Windows | Mac  |
| ------------- | ----------------- | ------ | ------- | ---- |
| v0.1 (Alpha)  | 鉄道の経路計算機能  | Alpha  | Alpha   | None |
| v1.0 (Stable) | 道路対応           | Stable | Stable  | None |
